# Voggel et al. (2020) — Key Reference

**Title:** The Nature of Ultra-compact Dwarfs Revealed by Gaia

**Authors:** Voggel, K. T.; Seth, A. C.; Neumayer, N.; et al.

**Journal:** The Astrophysical Journal, Volume 899, Issue 2, id.140

**Year:** 2020

**ADS Bibcode:** 2020ApJ...899..140W

**DOI:** 10.3847/1538-4357/aba842

**arXiv:** 2007.08557

**VizieR Catalog:** J/ApJ/899/140

## Significance

This is the primary methodology paper that inspired the BubbleTea search pipeline.
It demonstrates that Gaia's astrometric excess noise (AEN) can be used to identify
UCDs as marginally resolved sources, providing 689 UCD candidates around
Centaurus A (NGC 5128).

## Key Selection Criteria (from this paper)

- AEN > 0.5 mas
- BP-RP color: 0.8-1.8
- G magnitude: 16-21
- Proper motion filtering (permissive)
