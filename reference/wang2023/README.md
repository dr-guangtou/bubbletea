# Wang et al. (2023)

**Title:** Searching for Globular Clusters in the Outer Halo of M31 Using Gaia EDR3

**Authors:** Wang, S.; Ma, J.; et al.

**Journal:** The Astrophysical Journal, Volume 954, Issue 2, id.206

**Year:** 2023

**ADS Bibcode:** 2023ApJ...954..206W

**DOI:** 10.3847/1538-4357/ace963

**VizieR Catalog:** J/ApJ/954/206

## Significance

Demonstrates Gaia-based search for globular clusters (GCs) in the outer halo of M31
using Pan-STARRS photometry and machine learning classification. Relevant methodology
for BubbleTea's approach to identifying compact stellar systems at extragalactic
distances.
