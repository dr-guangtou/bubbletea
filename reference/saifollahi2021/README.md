## Table Roles and Reproduced Counts

- `tablea1.dat` contains 648 spectroscopically confirmed Fornax compact systems,
  mixing UCDs and globular clusters. The paper's 61 reported reference UCDs are
  reproduced exactly by `gmag <= 21` and a measured `0 <= rh < 75 pc`.
- `tablea5.dat` contains a classifier-selected pool of 1,155 objects: 222 objects
  named as UCD candidates and 933 named as globular-cluster candidates. This table
  is stored separately from canonical positive-reference records.
- `tablea6.dat` contains the final 44 visually cleaned, high-probability "BEST"
  UCD candidates. All 44 have exact-coordinate links to rows in `tablea5.dat`.

The paper reports that the 44 candidates still require follow-up spectroscopy.
Neither membership in table A5 nor table A6 is treated as confirmation.
