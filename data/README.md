# Data Directory

This directory contains all data products for the BubbleTea project, organized by
their role in the research pipeline.

## Structure

```
data/
├── external/               # Large catalogs (gitignored, accessed via env var)
├── gaia_queries/           # Cached Gaia query results
├── galaxy_sample/          # Target galaxy sample
│   └── galaxy_sample_ranked.csv  # 2,155 galaxies ranked by priority
├── literature/             # UCD literature database and catalogs
│   ├── catalogs/           # Original and matched UCD catalogs
│   │   ├── canonical_gaia_dr3_crossmatch.csv
│   │   ├── canonical_legacy_survey_dr10_crossmatch.csv
│   │   ├── all_ucds_*_matched.csv  # Superseded historical products
│   │   └── by_source/      # Per-paper CSV catalogs
│   ├── database/           # SQLite database for UCD compilation
│   │   ├── ucd_collection.db       # Immutable legacy input
│   │   └── ucd_reference_v2.db     # Rebuilt locally; gitignored
│   ├── discovery/          # Reproducible ADS search and screening tables
│   ├── sources/            # Paper metadata and ingestion plans
│   │   ├── key_papers.json
│   │   └── vizier_inventory.json
│   └── validation/         # Non-destructive literature audit products
└── results/                # Analysis results
    ├── background/         # Background density characterization
    ├── crossmatch/         # Candidate cross-match results
    └── radial_search/      # Radial profile excess detection results
```

## Provenance

| File/Directory | Source / Provenance |
|----------------|---------------------|
| `data/literature/database/ucd_collection.db` | Immutable legacy compilation of 2,108 rows from 9 database sources; see the validation report. |
| `data/literature/database/ucd_reference_v2.db` | Non-destructive v2 rebuild from the legacy database and authoritative raw tables; generated locally and gitignored. |
| `data/literature/catalogs/by_source/` | Individual catalogs downloaded from VizieR or provided by authors. |
| `data/literature/catalogs/canonical_gaia_dr3_crossmatch.csv` | Gaia DR3 rows for all 963 canonical source-ID associations; the complete target audit and manifest are under `validation/` and `sources/`. |
| `data/literature/catalogs/canonical_legacy_survey_dr10_crossmatch.csv` | Spherical two-arcsecond Legacy Survey DR10 enrichment for 3,723 canonical targets; the 4,359-target audit preserves no-matches and ambiguity. |
| `data/literature/discovery/` | ADS query results and a screening worksheet generated by `scripts/phase1_literature/search_literature.py`. |
| `data/literature/validation/` | Read-only audit, v2 validation, review queues, raw-file inventory, and exact-coordinate memberships. |
| `data/literature/sources/gaia_dr3_association_sources.csv` | Authoritative Gaia DR3 rows for the 117 shared-source identity groups; the companion JSON records the exact TAP query and SHA-256. |
| `data/galaxy_sample/galaxy_sample_ranked.csv` | Unified sample from LVG, CF4, NED-LVS, and SGA-2020. |
| `data/literature/sources/` | Current retrieval, screening, association, and cross-match manifests. `key_papers.json` and `vizier_inventory.json` are explicitly marked as superseded historical plans. |

## Verification Status

- **Legacy UCD Database**: Immutable 2,108-row comparison input from nine sources.
- **Galaxy Sample**: Verified 2026-03-27. 2,155 candidate host galaxies (D < 50 Mpc).
- **Literature v2**: Validation passes with no open Stage 1 gates. The current
  build preserves 5,049 literature records and 4,359 canonical objects: 740
  confirmed, 1,515 candidate, 2,082 rejected, and 22 uncertain. Four coordinate-
  null Fahrion provenance records remain explicit identity reviews. See
  `data/literature/validation/literature_reference_v2_validation.md` and
  `data/literature/validation/project_status_counts.json`.
